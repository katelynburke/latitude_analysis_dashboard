Summary: Latitude vs. X

The purpose of ‘Lattitude’ (latitude with attitude) was to analyze how particular weather changes as you become closer to the equator. I utilized CityPy (provides geo-coordinates of every city) and the OpenWeatherMap API to select over 500 random cities and perform weather checks on each city. I organized the raw data into an easy-to-read dataframe using Pandas.

MatPlotLib allowed me to create beautiful plots that show the various weather aspects vs. the city latitude. The weather trends that were analyzed from over 500 cities were: temperature, humidity, cloudiness, and wind speed. On this site, you will find the source data and visualizations created as part of the analysis, as well as explanations of trends and correlation witnessed.
